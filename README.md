# Marley Traders

Institutional-grade trading intelligence platform built for professional day trading.

## What it does
- Real-time multi-instrument market analysis
- Session-based trading intelligence (London, New York, Asia)
- Level monitoring with approach and tap detection
- Multi-timeframe T-Rex oscillator cascade
- Swing structure and liquidity analysis
- Live intelligence feed with agent-based signals

## Stack
- Python backend with FastAPI
- React + Next.js frontend
- Supabase database
- Docker deployment

## Status
Platform in active development. Public release coming soon at marleytraders.com
